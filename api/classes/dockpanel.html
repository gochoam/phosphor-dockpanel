Class DockPanel

A widget which provides a flexible docking area for content widgets.

Notes

Widgets should be added to a DockPanel using one of the dedicated dock methods. The inherited <prefix>Child methods should not be used. Widgets can be removed from a DockPanel by setting their parent reference to null.
